# Rey Alfred Bacornay Portfolio

A responsive personal portfolio website showcasing Rey Alfred Bacornay's IT skills, experience, projects, education, and contact information.

## Features
- Responsive design with mobile navigation
- Dark theme with modern styling
- Sections: About, Experience, Projects, Organizations, Skills, Contact

## Deployment

This portfolio now includes a PHP contact form and requires PHP hosting to use the form feature.

### Host as PHP site

1. Place the project folder in your PHP server root (for XAMPP, use `htdocs`).
2. Open `http://localhost/portfolio/index.php`.
3. The form sends email to `bacornayr@gmail.com` and logs submissions to `contact_submissions.txt`.

### Optional GitHub deployment

You can still store this project on GitHub, but GitHub Pages does not execute PHP. To host the contact form, use a PHP-capable host such as XAMPP, a shared PHP host, or a cloud provider with PHP support.

## Local Development

Served locally using XAMPP by placing the folder under `htdocs`, or any PHP-compatible web server.
